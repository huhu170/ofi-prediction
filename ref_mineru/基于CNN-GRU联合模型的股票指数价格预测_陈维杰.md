# 0 引言

股票市场的复杂性体现在价格是上市公司所有可获得信息的综合反映，其中包括公司的运营能力、财务状况、成长性等内部信息，也包括宏观经济走势、行业发展周期、政策变化等外部信息。从数据分析角度看，股票的定价可以看作是将高维特征压缩为二维价量信息的过程。如何从简单的成交信息中还原背后的影响因素，并据此指导未来价格的预测，是金融研究中具有挑战性的问题。传统的时序分析技术，如自回归移动平均（autoregressive moving average,ARMA）、自回归条件异方差（autoregressive conditionalheteroskedasticity, ARCH）等，都仅从价格序列本身寻找内在规律，难以真正实现价格的有效预测。通过引入卷积网络与循环网络等深度学习技术，预测模型可以结合更多外部的特征变量进行学习训练，并从数据中推理挖掘出隐藏的高维属性，有助于提高模型的实用性与准确率。

# 1 文献综述

股票以及股票指数的价格预测在本质上属于时间序列数据的建模与回归问题，即利用已获得历史与当前序列信息，通过寻找数据中的统计规律，对系统的未来形态进行预测。传统的时间序列建模技术包括线性回归、ARMA、ARCH 以及相关变体如 ARIMA、SARIMA、GARCH 等。

吴玉霞等利用 ARIMA 模型对创业板股票的价格走势进行预测，分析了不同阶数对模型精度的影响，并指出ARIMA(3,1,1) 模型对股价的短期动态与静态的预测准确率较高 [1]。方坷昊等在 ARIMA 模型的基础上，引入偏最小二乘法对股票指数进行回归分析，能够降低数据自相关性的影响，修正原有模型的估计误差 [2]。方燕等分析了股票的尖峰厚尾分布特性，通过结合ARIMA与GARCH的优势建立了联合模型，并对传媒板块的股票价格进行预测，结果表明联合模型在提高模型精度的同时，也能够较好地克服时序数据的自相关性与异方差性问题 [3]。

上述模型的核心问题在于模型的预测结果主要依赖于序列本身的规律，通过过去的价格推导未来，难以充分利用其它变量带来的信息。同时，股价数据的高波动性以及噪声干扰也会导致较大的估计误差，严重降低模型的预测稳定性。近年来，随着机器学习、深度学习技术的发展与成熟，以及硬件计算能力的提升，模型的特征表达能力以及数据处理能力得到了较大程度的拓展，各类网络模型也被应用到股价预测中，包括人工神经网络（artificial neural net-work, ANN）、循环神经网络（recurrent neural network,RNN）、卷积神经网络（convolutional neural networks,CNN）、长短时记忆网络（long short term memory net-work, LSTM）等。

潘水洋等利用 ANN 构建了股票的定价预测模型，以市场组合收益率、市值以及账面市值比作为解释因子，对股票收益率进行预测，结果表明比传统的 Fama-French 三因子定价模型效果更优 [4]。黄丽明等利用 RNN 与自然语言处理技术，对股票的历史价格以及相关新闻的文本信息进行训练，能够融入更丰富的数据信息，提高预测的准确率 [5]。宋刚等采用LSTM 作为基础模型，结合粒子群算法对网络结构的超参数进行寻优，增强模型的普遍适用性[6]。

然而，股票中的交易数据已经是被高度压缩后的结果，尤其是在短期预测中，成交价格与成交量几乎是唯一可获得的数据信息，单一的网络结构难以直接从历史价量中建立与预期价格之间的关系，因此需要考虑利用不同的网络结构，结合各自的优势特点进行联合建模。

# 2 CNN-GRU 联合模型

针对股票指数价格的预测问题，本文建立了 CNN-GRU 联合模型，利用 CNN 的特征提取能力，从原始数据中自动挖掘出深层隐藏特征，同时利用 GRU 兼顾股票数据的时序性，提高模型的预测精度，最后通过三只股票指数的实际数据进行验证。

# 2.1 卷积神经网络

卷积神经网络 CNN 是一种通过多个检测层进行特征提取并训练的前馈神经网络 [7]。CNN 能够在不提供显式规则与标注的前提下，隐式地从数据中进行抽象学习，高效地从数据中提取特征，因此被广泛应用到计算机视觉、语音识别、自然语言处理等领域 [8]。一个完整的 CNN 结构通常由卷积层、激励层、池化层三部分组成。

CNN 的主要功能在于以数据驱动的形式训练不同的滤波器，即卷积核，从而捕捉原始数据在不同角度下的抽象特征，因此卷积层是 CNN 网络结构的核心。卷积层利用卷积核对输入层进行遍历扫描，完成对输入层的数据转换。设输入层为$N { \times } N$ 的矩阵，卷积核大小为 $F { \times } F _ { i }$ ，卷积后的输出值可由式（1）计算，有：

$$
\nu _ { i , j } = f ( \sum _ { k = 0 } ^ { F - 1 } \sum _ { m = 0 } ^ { F - 1 } w _ { k , m } V _ { i + k , j + m } )
$$

式中： $\nu _ { i , j }$ 为输出矩阵在第 $i$ 行第 $j$ 列的输出值， $w _ { k , m }$ 为卷积核中第 $k$ 行第 $m$ 列的权重， $V _ { i + k , j + m }$ 为输入层在第 $i { + } k$ 行第 $j { + } m$ 列的原始值， $f$ 为激励函数。经过从上至下，从左至右的遍历卷积运算后可得到 $E \times E$ 的输出矩阵，其中 $E { = } N { - } F { + } 1$ 。

激励层通过类似于传统神经网络的激活函数，对卷积计算后的输出值进行非线性变换，从而增强网络的表达能力。常用的激励函数包括 ReLU、Sigmoid、tanh 等。

池化层通过对卷积后的数据进行重新采样，在保证特征不变性的条件下去除冗余特征，只保留最具代表性的特征，从而获得更概括的表达。池化层在降低数据维度的同时，也减小了局部数据变动对模型结果的影响，使得模型更稳健并避免过拟合现象。

典型的 CNN 网络主要用于处理图像数据，输入样本具有长度与宽度二维特征。针对股票价格等时间序列数据，每个样本只具备一维特征，因此需要从时间维度进行拼接，组成二维矩阵，并采用 CNN 的时序版本 Conv1D 进行建模。

通过对输入层施加多个卷积核进行运算，可获得关于原始数据的不同特征表达。不同特征之间可以独立并行运算，加快模型的学习速度。同时，CNN 在卷积核内神经元权重固定的参数共享机制也有效降低了网络的复杂度，提升模型的训练效率。

# 2.2 门控循环单元

门控循环单元 GRU 是一种用于优化时序数据长期记忆的循环神经网络模型 [9]。为解决传统 RNN 在反向传播中出现的梯度消失与梯度爆炸问题，LSTM 引入了输入门、遗忘门、输出门等门结构对历史数据进行过滤与筛选，从而提高模型对长序列数据的记忆能力 [10]。GRU 则是在 LSTM 的基础上进一步优化，将三个门结构合并为重置门与更新门，获得更简练的表达，研究表明 GRU 在多种时序预测任务中都能获得比LSTM 更好的表现。

重置门用于控制历史数据中有多少需要保留以及最新数据中有多少需要被引入，负责对时序数据的短期形态进行记忆。更新门用于控制有多少数据需要被丢弃，负责筛选序列中不重要的信息。在给定当前时刻的输入值 $x _ { t }$ 后，重置门状态 $r _ { t }$ 与更新门状态 $z _ { t }$ 可由式（2）、（3）计算，有：

$$
r _ { t } = \sigma ( x _ { t } w ^ { r } + h _ { t - 1 } W ^ { r } + b ^ { r } )
$$

$$
z _ { t } = \sigma ( x _ { t } w ^ { z } + h _ { t - 1 } W ^ { z } + b ^ { z } )
$$

式中： $w ^ { r }$ 、 $W ^ { r }$ 、 $w ^ { z }$ 、 $W ^ { z }$ 为重置门与更新门的权重系数矩阵，$b ^ { r }$ 、 $b ^ { z }$ 为各自的偏置量， $h _ { t - 1 }$ 为上一时刻的隐藏状态， $\sigma$ 为Sigmoid 激励函数，可将输出值映射到 $0 \sim 1$ 范围，作为控制门输出比例的信号。

获得重置门与更新门状态后，计算临时状态 $h ^ { \prime }$ 并通过与上一时刻的隐藏状态 $h _ { t - 1 }$ 进行调和，最终获得当前时刻的输出状态，有：

$$
h ^ { \prime } = \operatorname { t a n h } ( x _ { t } w ^ { h } + ( h _ { t - 1 } \odot r _ { t } ) W ^ { h } + b ^ { h } )
$$

$$
h _ { t } = ( 1 - z _ { t } ) \odot h _ { t - 1 } + z _ { t } \odot h ^ { \prime }
$$

式中： $\boldsymbol { w } ^ { h }$ 、 $W ^ { h }$ 、 $b ^ { h }$ 为临时状态的权重系数矩阵与偏置量，tanh 为激励函数， $\odot$ 表示矩阵点乘。对于时序数据，最后时刻的 $h _ { t }$ 将被作为GRU的输出传递到网络的下一层。

如前文所述，由于股票价格及其特征是一维数据，所以在网络的输入端需要将数据拼接为二维结构，其中行表示时间长度，列表示特征维度，并在建模时采用一个 batch 长度的样本进行批次训练。随后将原始数据输入 CNN 层，利用多个尺寸相同的卷积核完成卷积操作，提取原始数据在不同维度下的特征信息，并将输出结果拼接成一个二维矩阵。该矩阵可看作是经 CNN 转换后的新的特征表达，其中行表示时间长度，列表示新的特征维度，新特征的维度等于卷积核的个数。

为减小过拟合风险，所提出的模型还在 CNN 层后引入了Dropout 层，每次随机屏蔽若干个神经元，避免模型过分依赖某些特征，提高模型的稳健性。经过滤后的新特征将作为输入传递给 GRU 层，据此训练数据中的时序规律。GRU 层取时间序列中最后时刻的隐藏状态作为输出，并通过全连接层加权求和获得最终的输出值，即下一时刻的股票预测价格。

# 4 结论

本文针对股票指数价格的预测问题，结合卷积神经网络与门控循环单元的各自特点，建立了 CNN-GRU 联合模型，并通过上证指数、沪深 300、中证 500 三只典型的股票指数进行验证。模型在传统技术面构造特征因子的基础上，利用CNN 进一步挖掘数据中的高维信息，并利用 GRU 完成时间序列规律的训练。测试结果表明，所建立的模型在 RMSE、MAE以及方向准确率 $\mathrm { D } _ { \mathrm { { s t a t } } }$ 等指标下均获得较高的精度，并具有较好的时效性与稳定性。相关结果验证了深度学习在金融时间序列的建模与预测能力，为投资者与量化研究人员提供参考。
