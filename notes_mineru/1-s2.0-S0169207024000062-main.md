## 3）主题笔记（Topic Notes）
### OFI Definition and Application in Deep Learning Models
依据证据 E1、E2：OFIs are calculated as the difference between bid and ask order flows at the first order book level（E1）. The deepOF model leverages OFI as a stationary input to the deepLOB architecture, improving prediction performance over raw order book data（E2）. This makes OFI a key feature for short-term return prediction of NASDAQ stocks.

### Volume Representation for Robust Predictions
依据证据 E5、E7：The volume representation（deepVOL）meets five robustness criteria（region of interest, efficiency, validity, smoothness, compatibility）that raw order book or OFI-based models do not（E5）. It performs comparably to OFI-based models（deepOF）with L2/L3 data, being in the top-performing set for most predictable horizons（E7）.

### Predictability of High-Frequency Returns
依据证据 E4、E6：High-frequency mid-price returns of NASDAQ stocks show systematic predictability up to50 order book events ahead at the99% confidence level（E4）. This result is statistically validated using MCS p-values, ensuring it is not due to random variation（E6）.

### Limitations of Mid-Price Based Predictions
依据证据 E8：Mid-price returns are non-tradable, which limits the practical application of the study's predictive insights to real-world trading strategies（E8）. This is a critical consideration for translating academic findings into actionable trading signals.
