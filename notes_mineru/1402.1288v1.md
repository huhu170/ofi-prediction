## 3）主题笔记（Topic Notes）
### Permanent Market Impact: Linear Assumption and Justification
The permanent market impact (PMI) of a metaorder is defined as the limit of the impact function as time tends to infinity (E6). The paper justifies that PMI should be linear in the metaorder volume under the mechanical impact assumption (E1). This linearity comes from functional arguments (additivity of impact for split executions) and no-price-manipulation arguments (E1).

### Price Dynamics from Order Flow: Martingale Assumption and Impact Equation
Using the linear PMI assumption and a martingale hypothesis on the price (Assumption3), the paper derives an equation linking price dynamics to the expected future order flow imbalance (E2). This equation resolves the paradox of persistent order flow and martingale prices by framing price changes as surprises in order flow (E2).

### Hawkes Process Model for Order Flow and Impact
The paper uses Hawkes processes to model order flow, which captures the persistence (apparent long memory) in order flow (E8). For this model, the price process is a propagator model with kernel ζ linked to the Hawkes kernel φ (E7). The impact function of a metaorder in this model is asymptotically a power law (E3), consistent with empirical observations like the square root law.

### Link Between Impact Exponent and Long Memory of Order Flow
The exponent of the impact power law (ν) is linked to the long memory exponent of the order flow (γ) via ν=(1+γ)/2 (E4). This link is derived from the Hawkes model's properties, where the long memory of the order flow comes from nearly unstable Hawkes processes (E4, E8).

### Limitations of the Hawkes Model
The Hawkes order flow model used in the paper is simplistic (E5). The paper suggests that more realistic models, such as those where order flow is a superposition of independent metaorders (like [32]), are needed to better capture real-world order flow dynamics (E5).
